Can I host this anywhere?

Yes. It’s a static file that works on GitHub Pages, Netlify, Vercel, S3/CloudFront, and more.

How do I integrate my form?

Point the onsubmit to your API, or use services like Formspree, Make, Zapier, or webhooks.

Does it support analytics?

Insert your provider’s script (GA4, Plausible, PostHog, etc.) before </body>.

Is it accessible?

Yes — proper semantics, labels, ARIA states, and keyboard navigation.
